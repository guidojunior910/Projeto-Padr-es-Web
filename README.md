# Projeto: React em 9 tópicos — Como compartilhar

Para permitir que outra pessoa visualize o site, siga um destes passos:

- Abra o arquivo `index.html` no navegador (duplo clique ou arraste para uma aba do navegador).
- Compacte a pasta inteira `Projeto.html` e envie o .zip — o destinatário deve descompactar e abrir `index.html`.
- Publique no GitHub Pages: crie um repositório, envie os arquivos e ative o GitHub Pages apontando para a branch `main` (ou `gh-pages`).

Observações:
- Os arquivos usam caminhos relativos para `Projeto.css` e `Projeto.js`; mantenha esses arquivos no mesmo diretório de `index.html`.
- Se houver imagens ou outras dependências, inclua-as na mesma pasta para manter as referências.

Se quiser, eu posso: compactar a pasta para você, mover todos os arquivos para uma pasta `site/` ou gerar um passo-a-passo para publicação no GitHub Pages.
